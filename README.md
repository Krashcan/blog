*	Speedy blog 遵循GPL-3.0条例开源 基于 revel + xorm 为基础搭建
*	使用mysql／mariadb + redis作为持久层
*	此项目正在开发中，欢迎加入一起开发

## 快速体验
  
&#12288;&#12288;导入SQL文件（目前开发使用的是mysql／mariadb），配置/conf目录下的speedy.conf文件 <br/>
将Redis和mysql／mariadb 地址以及相关修改成自己本地的。执行 ``` $ ./revel run blog ``` 即可
  

## 依赖

*	revel
*	xorm
*	redis.v5
